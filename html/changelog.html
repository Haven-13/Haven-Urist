12 July 2021

martinlyra updated:

  • Fixed an incorrect typepath affecting how the overmap generators place events
  • Ported tgUI 4.3 from /tg/-station
  • The engineering & janitorial tool carts UIs will tell you what you can fit in them
  • The biogenerator now show its current buffer in the UI
  • Added feature to order batch processing jobs at biogenerators
  • Deleted old nanoUI
  • Deleted UI from fire alarms. Now you have to do now is to click on them to (de)activate the area alarm.
  • Deleted superfluous beaker from biogenerators
  • Deleted superfluous waypoints & bookmarks from helm computers
  • Fixed ship helm computers offering a quick way to commit Jonestown-style mass-suicide - (Deleted Auto-pilot)
  • Entirety of the interface is now dark-themed with Dwarf Fortress and Cataclysm Dark Days Ahead-inspired colouring schemes for text messages.
  • Most of the new code now use the tgUI instead of nanoUI. Meaning that lot of UI will be missing.
  • Janitorial & Engineering carts rewritten from scratch
  • Sleeper UI redesigned from scratch
  • Ship Helm UI redesigned from scratch

25 June 2021

martinlyra updated:

  • Fixed wall-frame stripes
  • Fixed the white floor decals, in the operating theatres, being of wrong colour
  • Changed the old IV-drip to a new one, with bigger status display and additional states for when no beaker is attached.
  • Added support for mutable_appearance
