25 May 2021

martinlyra updated:

  • Removed integrated circuits completely
  • Fixed stability issue related to first-time players and preferences
  • The lobby screen menu now updates when the round has started

16 May 2021

martinlyra updated:

  • Added static overmap objects, currently it is just stars for now
  • Added new natural planetary system generation for overmap
  • Removed obsolete unused exoplanet features
  • Changed `ceil` to `Ceiling`
  • Added Visual Studio Code settings files

09 May 2021

martinlyra updated:

  • Added a special null turf to better handle unused turf space
  • Malfunctioning AI is no longer votable
  • Removed Borer gamemode
  • Removed Changeling gamemode
  • Removed Cult gamemode
  • Removed Deity gamemode
  • Removed Ninja gamemode
  • Removed Vampire gamemode
  • Removed Wizard gamemode
  • Supermatter cascade is no longer possible
  • Removed the Baycode custom items feature
  • Removed synthesizing instruments
  • Removed unused sound files

07 May 2021

martinlyra updated:

  • Added a second mode to the starlight config, allowing dynamic lighting to be enabled for all space turfs.
  • Lighting effect now uses alpha to improve its rendering effect
  • Lighting calculations switched from subtractive to additive
  • Moved starlight function to its own file in lighting module
  • Added preliminary integration with auxtools

06 May 2021

martinlyra updated:

  • The old Baystation Open-Space has been fully deleted. Good bye. You won't be missed.
  • Fix a large initialization overhead caused by floors turning into plating then back to their given flooring type
  • Fix runtimes related to lighting brightness
  • Fix problems related to grid floors
  • Fix layering issue with above-light-effects on light-switches and robots
  • Most of the old multi-z updating code has been minimized to decrease overhead
