04 October 2021

martinlyra updated:

  • Removed the fountain of youth
  • Removed the temple from the orb asteroid
  • AWAIT changed to AWAITING in nuclear bomb UI
  • Blocking procs and calls have been forbidden from garbage collection procs

02 October 2021

martinlyra updated:

  • Added tgUI for portable generators
  • Fixed runtimes caused by improper garbage collection of unused map-views.
  • The stationary self-destruct terminal accepts, and requires the nuclear authorization disk.
  • Fixed the nuclear bomb and self-destruct UI

26 September 2021

martinlyra updated:

  • The preview is now tgstation-based instead of using a static image. Using a map for easy, clean, and reliable updating.
  • There are now three different backgrounds to select for the preview instead of four.
  • The shadow-people have been sent back to the shadow realm from the character creator preview.
  • As of the above tweak, the delay that causes a discrepancy between client and server has been removed.
  • Fixed a runtime preventing the old HTML UIs from responding properly to input
  • Added the Synthmorph culture, the default culture for the Machine/IPC species on the Pluto map

retlaw34 updated:

  • Redoes the fuel port and oxygen wall locker

15 September 2021

martinlyra updated:

  • A popup text at top of screen will appear when howevering over an atom
  • Preference (in Global category) allowing players to show or hide the popup text. It is turned on by default.
  • Preference (in Global category) controlling whether the hover-over should display in either the stylic or the San Serif font.
  • Byond interface CSS is now compiled and loaded at runtime instead of compile-time

13 September 2021

martinlyra updated:

  • Added potential bugs related to map UI elements
  • Renamned typepaths for UI elements to a special atom/movable instead for technical imporvements
  • Localhosts now will be automatically given a total-control rank, it is enabled by default in the config, but false in code
  • Localhosts now can bypass guest-key checks as long the automatic rank is enabled
  • Admins now bypass guest and whitelist checks for jobs

08 September 2021

martinlyra updated:

  • Fixed several cases of mangled characters showing up in text and some interfaces

27 August 2021

martinlyra updated:

  • Optimized away maps
  • Changed how objects initialize to remove time spent on waiting, to cut down the times.
  • Chemistry reactions are now handled by a small subsystem for the time being
  • Fixed cave & mineral generation not being synchronized with the server & mapping initialization
  • Fixed overmap objects whose location are known not showing up
  • Fixed star not showing up.

26 August 2021

martinlyra updated:

  • Added overlays system
  • Removed vis_overlays system
  • Changed how airlocks behave when animating
  • Changed emissives from using 3 planes to 1 plane for efficiency and performance boost.

25 August 2021

martinlyra updated:

  • Compiler options moved to code root folder from __defines folder.
  • Compiling toolchain has been changed from simple node building to a juke-build suite.

23 August 2021

martinlyra updated:

  • Developer QoL: Whitespaces in source code will now be CI-checked. This will not affect the game directly.

10 August 2021

martinlyra updated:

  • Added missing UI for escape pod control
  • Added missing UI for escape pod berth control
  • Fixed the evacuation selection not properly annotating which option was selected in the Communications program

05 August 2021

martinlyra updated:

  • SMES units now have emissive overlays
  • Added new icons for the SMES units, based on the old one

03 August 2021

martinlyra updated:

  • Added tgstation emissive overlays
  • Added tgstation visual overlays (overlays that use vis_contents)
  • Solar control computer uses board-computer update_icon now
  • Fixed the power monitoring being a snowflake object, it should be more uniform with other board computers
  • Both solar and power computers have been converted to screen overlays.
