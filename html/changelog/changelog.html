martinlyra updated:

  • SMES units now have emissive overlays
  • Added new icons for the SMES units, based on the old one

03 August 2021

martinlyra updated:

  • Added tgstation emissive overlays
  • Added tgstation visual overlays (overlays that use vis_contents)

  • Fixed the power monitoring being a snowflake object, it should be more uniform with other board computers
  • Both solar and power computers have been converted to screen overlays.

martinlyra updated:

  • Fix a bug related to the ferry-type shuttles not giving the right values on wherever its current location was.

29 July 2021

martinlyra updated:

  • Added proper evacuation protocols to Pluto, it should be fully functional now
  • Added the required docking controllers for the bottom deck lifepods
  • Changed the lifepod airlocks to glass hatches on Pluto
  • Changed most of the unused space tiles to null turfs in Pluto's "remote" location to reduce the impact on performance
  • Made the indentation in the DM code more consistent
  • Changed direct uses of << to corresponding macros depending on the context and use, actual bitshifts are untouched
  • Changed direct uses of >> to `from_file`, actual bitshifts are untouched

  • Fix errors in Auto-CL generation

17 July 2021

  • The changelog now uses the new asset delivery system
  • Fixed an asset delivery issue that prevented logos to show up proper on paper
  • Fixed an asset delivery issue that did not supply the NtOS programs the header icons they needed

15 July 2021

martinlyra updated:

  • Fixed an pass-by-reference issue that deleted all hairs and facial hairs from species lists when using the appearance changer
  • Added underscores to lazy-access macro names
  • Removed Unathi's regeneration ability
  • Removed the "Make pAI" verb, as there are simpler ways to do it
  • Removed kick mine
  • Lowercased names of all mines so far
  • Radiation mines now respect protection values against radiation
  • All varieties of land mines can now be picked up, armed own, or disarmed.

13 July 2021

martinlyra updated:

  • Added CI testing for away maps, to ensure they are bug-free

12 July 2021

martinlyra updated:

  • Fixed an incorrect typepath affecting how the overmap generators place events
  • Ported tgUI 4.3 from /tg/-station
  • The engineering & janitorial tool carts UIs will tell you what you can fit in them
  • The biogenerator now show its current buffer in the UI
  • Added feature to order batch processing jobs at biogenerators
  • Deleted old nanoUI
  • Deleted UI from fire alarms. Now you have to do now is to click on them to (de)activate the area alarm.
  • Deleted superfluous beaker from biogenerators
  • Deleted superfluous waypoints & bookmarks from helm computers
  • Fixed ship helm computers offering a quick way to commit Jonestown-style mass-suicide - (Deleted Auto-pilot)
  • Entirety of the interface is now dark-themed with Dwarf Fortress and Cataclysm Dark Days Ahead-inspired colouring schemes for text messages.
  • Most of the new code now use the tgUI instead of nanoUI. Meaning that lot of UI will be missing.
  • Janitorial & Engineering carts rewritten from scratch
  • Sleeper UI redesigned from scratch
  • Ship Helm UI redesigned from scratch
  • Reorganized some files in the source code tree
  • Rebranded from AstroHaven-13 to Haven-13

25 June 2021

martinlyra updated:

  • Fixed wall-frame stripes
  • Fixed the white floor decals, in the operating theatres, being of wrong colour
