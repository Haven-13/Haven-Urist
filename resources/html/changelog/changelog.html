martinlyra updated:

  • Modular computers' UI-state changed from physical-only to default
  • Fixed AI being unable to interact with all physical objects that can be accessible by the AI:
  • Fixed some mob icon updates getting deferred too late instead of being done immediately, should see less blank images now.
  • Replaced the old ship sprite with Pawn's ship sprite for overmap
  • Added Pawn's shuttle sprite for overmap
  • Added Ion Drives for use with overmap-capable shuttles
  • Added overmap flight for Pluto's shuttles and future ones
  • Readjusted the vessel mass of Styx and Eris shuttles
  • Fixed a bug where shuttles would act as control and helms for the mothership
  • Pluto's Eris and Styx shuttles are overmap capable, finally
  • Added premliminary art for shuttles on the overmap
  • Added sprites for the Ion Drives
  • Fixed AI-vision obfuscation layering issue
  • Added metrics from Paradise for profiling and analysing performance without interfering the server itself
