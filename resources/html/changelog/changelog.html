04 February 2022

Lohikar, Semoro, and azizonkg updated:

  • Updated the timer subsystem to latest critical Baystation12 code.
  • Optimized the timer performance
  • Improved the reliability of the timer subsystem and its logic

02 February 2022

martinlyra updated:

  • Fixed a priority issue with background ticker subsystems
  • Fixed permission check issues with UI where ghosts would be able to interact UI unrestricted
  • Fixed users never leaving fingerprints on objects they interact with via UI due to improper use of the interaction proc

01 February 2022

martinlyra updated:

  • Added the missing RCON program UI
  • Addressed a security issue related to dependency in tgUI
  • Added the missing UI for airlocks being controlled by silicons
  • Fixed AI not being able to "see" objects despite player vision
  • Added UI for the Shield Monitoring program
  • Changed the toggle button on the shields' mode settings to larger checkbox buttons
  • Fixed the Shield Monitoring program's UI being missing among with few immediate fixes to the UI data and interaction

29 January 2022

martinlyra updated:

  • Fixed the main elevator using the wrong turf for the bottommost deck on Pluto
  • Due to outdated code, the projectile effect for the ninja net thrower has been disabled
  • Fixed mining turfs showing up as bright when they should not
  • Fixed projectiles not showing up properly
  • Optimized how global variables and information are handled by the game.
  • Globals, statics, and constants are now unit-tested for sanity
  • Added the missing UI for gas tanks
  • Fixed the UI being missing for all gas tanks that can be activated in hand
  • Show Server Revision now shows both the client's and server's full version
  • Updated the TGS host to attempt fix withstanding issues with the version information and more features.
  • Fixed test-merge information not showing up properly
  • Fixed sever-revision info not showing up properly
  • Oxford Comma is now default for text from `english_list`, expect to see them in plenty of places
  • Fixed a runtime error related to vis_locs when viewed in View-Variables debug tool

24 January 2022

Glloyd, martinlyra, Naxodile updated:

  • Added a pirate station that periodically spawns pirate vessels to fight
  • Added more features to ship combat against NPC vessels
  • Added the datumized faction system from Urist
  • Added factions reputation and relation from Urist
  • Ghosts should be able to spawn as defenders on the pirate station upon the first breach
  • Datumized ship weapon ammunition
  • Player-controlled vessels should be able to engage in PvP ship combat, untested
  • Most Urist content now uses the datumized faction system
  • Ports revised burst firing mechanic for ship weapons
  • NPC ship shields are now generally tougher

22 January 2022

martinlyra updated:

  • Fixed atoms not being loaded correctly when spawned midgame.

13 January 2022

martinlyra updated:

  • Cleaned up away maps that were causing errors for the game.
  • Fixed a layering issue with disposal pipes.
  • Fixed initialization issues related to explosions spawned in map init.
  • Fixes the multi-z rendering issue, in which it would be permanently broken for a player the moment they enter a submap with incorrect Z ordering.

martinlyra updated:

  • Modular computers' UI-state changed from physical-only to default
  • Fixed AI being unable to interact with all physical objects that can be accessible by the AI:
  • Fixed some mob icon updates getting deferred too late instead of being done immediately, should see less blank images now.
  • Replaced the old ship sprite with Pawn's ship sprite for overmap
  • Added Pawn's shuttle sprite for overmap
  • Added Ion Drives for use with overmap-capable shuttles
  • Added overmap flight for Pluto's shuttles and future ones
  • Readjusted the vessel mass of Styx and Eris shuttles
  • Fixed a bug where shuttles would act as control and helms for the mothership
  • Pluto's Eris and Styx shuttles are overmap capable, finally
  • Added premliminary art for shuttles on the overmap
  • Added sprites for the Ion Drives
  • Fixed AI-vision obfuscation layering issue
  • Added metrics from Paradise for profiling and analysing performance without interfering the server itself
